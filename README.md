# TypingTempest

TypingTempest is a typing practice application focused on helping users improve accuracy and speed while typing source code or custom text passages.

Originally developed as a prototype in LiveCode, TypingTempest has been rewritten in **C#** as a Windows desktop application with enhanced editing, syntax highlighting, and speech features.

## Features

- Practice typing source code, scripts, or custom text content
- Real-time feedback while typing
- Syntax highlighting for improved readability
- Backspace support for correcting mistakes
- Lesson-based practice content
- Speech/audio cues for accessibility and learning support
- Modern Windows desktop experience built with C#

## Technology

TypingTempest is developed in **C#** using the .NET platform.
